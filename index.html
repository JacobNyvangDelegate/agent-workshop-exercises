Copilot Studio Workshop

Agent Building Exercises

Use these exercises to design, build, and test your agent step by step.

Purpose of the workshop

The purpose of these exercises is to build a complete Copilot Studio agent step by step. You will start with a simple assistant, add knowledge, design structured conversations, connect to business data, and trigger actions using flows.

By the end, you should understand how the core building blocks of an agent work together: instructions, knowledge, topics, Dataverse MCP, and agent flows.

Make sure you have completed the workshop setup before starting the exercises.
