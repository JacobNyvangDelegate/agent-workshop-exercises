Lab 0 · ~10 minutes

Workshop Setup

Before building agents, let's make sure everyone has access to the correct environment and can sign in successfully.

🚀 What you'll do

  • Select your assigned workshop user
  • Sign in to Copilot Studio
  • Configure MFA
  • Set English as your language
  • Select the correct environment

Step 1: Find your workshop user

  1. Open the workshop spreadsheet.
  2. Select an available user.
  3. Keep this user for the entire workshop.

Open Workshop User Assignment Sheet

Step 2: Download the workshop material

Download the workshop knowledge base before starting the labs. These files are used throughout the workshop as knowledge sources, test data, icons, and supporting material for the bakery agent.

  1. Download the workshop material.
  2. Extract the downloaded folder if it is zipped.
  3. Keep the folder available during the full workshop.

Download workshop material ↓

💡 Keep this folder available

You will need these files in multiple labs, especially when uploading knowledge sources and configuring the bakery agent. Save the folder somewhere easy to find.

Step 2: Open Copilot Studio

  1. Open a private/incognito browser window.
  2. Navigate to:
https://copilotstudio.microsoft.com/
  1. Sign in with your assigned workshop user.

Step 3: Complete first-time setup

  1. Configure Microsoft Authenticator.
  2. Change the temporary password if prompted.
  3. Set your preferred language to English.
  4. Select United States as the trial region if requested.

Step 4: Select your environment

  1. Open the environment selector.
  2. Locate the environment matching your assigned user.
  3. Select that environment.

Example:

User 5 → Environment 5 User 12 → Environment 12

Step 5: Import the workshop solution

  1. Download the workshop solution file.
  2. Go to Power Apps
  3. Open Solutions in the left navigation pane.
  4. Select Import solution in the top navigation bar.
  5. Upload the downloaded ZIP file.
  6. Follow the import wizard and complete the import.
  7. The import can take several minutes to complete.

Download workshop solution ↓

💡 Solution required for later labs

The workshop solution is required to complete Lab 5 and Lab 6 It contains the Dataverse table used later in the workshop.

Step 6: Verify your setup

Confirm the following before continuing:

  • ✅ You can access Copilot Studio
  • ✅ MFA is configured
  • ✅ Language is English
  • ✅ Correct envi ronment is selected

🎯 Checkpoint

  • ✅ Logged in successfully
  • ✅ MFA configured
  • ✅ Language set to English
  • ✅ Correct environment selected
  • ✅ Setup Solution for Sweet Bakery
  • ✅ Ready to start Lab 1
